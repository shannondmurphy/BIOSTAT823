Univariate Analysis on the Growth of COVID-19 Death Count

Univariate time series models can be AR, MA, ARMA, SARIMA, etc. The foundational components of these models are the autoregressive (AR) model and the moving average (MA) model. Both of these models are types of random processes.

Autoregressive models predict a variable value for a future time point, $X_{t}$, using past values ($X_{t-1}, X_{t-2},...,X_{t-p}$). The order of an AR(p) model, p, defines how many previous time points are used to predict the next value.

Moving average models predict a variable value for a future time point, $X_t$, using past error terms rather than past values of the variable. The order of an MA(p) model, p, defines how many past error terms are used to predict the next value.

An ARMA model uses both past variable values and past error terms to predict a future value. This is the kind of model used in the univariate section of this project.

Had there been evidence of seasonal trends, a SARIMA model would have been appropriate.

Multivariate Modeling

A VAR model is a multivariate version of an AR model. That is the multivariate model used in this project.

Three VAR models will be constructed and compared.

  • Model with only the features highly correlated with deaths
  • Model with only the features with low correlation to deaths
  • Model with all features

Because the univariate model chosen used an AR(1) component, we evaluated all our multvariate models with a lag of one week. Training and test data were split the same way they were for the univariate model.

These plots clearly favor the model using variables that are not highly correlated with deaths. This is likely the result of overfitting. We know that these variables are not correlated with deaths, so it doesn't make sense for them to be informative of the death trend.

The model using highly correlated data, I hyopthesized, would be the best model. However, it seems to have added more noise and not anymore informative information. Hospitalizations and Adult ICU COVID-19 patients are the two variables that were added. It may be the case that many people hospitalized or in the ICU do not end up dying of COVID-19, or the rate that those hospitalized or in the ICU die from COVID-19 is inconsistent, making the relationship difficult to define. Overall, the information was not helpful in predicting the next five weeks of data, and the univariate model was better.

The full model performed the worst of all the models. This is interesting because frequently adding more variables will improve models, even if it only results in overfitting. That was not the case here because the additional information created too much noise within the system. The model did not pick up the same information it was able to in any of the three models before.

Other Potential Exploration

  1. There are other methods for time series models. One popular method is an LSTM (long short-term memory) model, which is a recurrent neural network. It could be interesting to compare the results of an LSTM on this dataset with the results from our univariate and multivariate models reported here.
  2. It is clear that the ARMA and VAR models are not the best methods to forecast COVID-19 deaths. This is likely because COVID-19 deaths are not random events. They depend on interpersonal contact, transmission rate, death identification, etc. Epidemiological methods, such as an SIR model, would be more effective. There are many papers from the last year with epidemiological models to forecast COVID-19 deaths, cases, hospital bed usage, and other variables.
